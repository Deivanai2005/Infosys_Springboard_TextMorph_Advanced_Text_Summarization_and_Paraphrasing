# Infosys_Springboard_TextMorph_Advanced_Text_Summarization_and_Paraphrasing
This repository is created as a part of the Infosys Springboard Internship 6.0 program.  It contains project-related materials for the TextMorph – Advanced Text Summarization and Paraphrasing module.

**TextMorph**
AI-Powered Content Simplification, Summarization & Paraphrasing

Transform complex content into simple, clear, and readable text using AI.

# 📖 About

TextMorph is an AI-based tool that helps users:

Simplify complex text

Summarize long content

Paraphrase in different styles

Improve readability

📌 Developed for Infosys Springboard Internship
📌 Users: Students, Teachers, Researchers, Content Writers

# 🎯 Problem

Many people find complex text hard to understand

Manual simplification takes time

# ✅ Solution

TextMorph uses AI to:

Make text easy to read

Reduce content length

Rewrite content clearly

# 🚀 Features

👤 User Features

🔐 Login & Registration (JWT + OTP)

📊 Readability Score (Flesch, SMOG, etc.)

✂️ Summarization (Short / Medium / Long)

🔁 Paraphrasing (Simple / Neutral / Advanced)

👥 Compare Original vs Output

⭐ Feedback (rating + comments)

🕘 History (save & download)

👤 Profile Management

 Admin account

# 🧩 Architecture

Monolithic system

Backend + Database + AI models integrated

# Tech Stack

Frontend: Streamlit

Models: Hugging Face Transformers

Database: SQLite

Security: JWT, bcrypt

Deployment: Docker

# 🤖 Models Used

BART → Summarization + Rewriting

FLAN-T5 → Paraphrasing

Pegasus → Summarization

NLTK → Readability

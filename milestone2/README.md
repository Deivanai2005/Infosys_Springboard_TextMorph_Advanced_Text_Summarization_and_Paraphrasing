# Milestone 2 – Readability Dashboard with OTP Authentication

## About the Project
This project is an extension of Milestone 1 and focuses on improving security, usability, and visualization.
It includes OTP-based authentication to enhance user security and a Readability Dashboard that analyzes
text and displays readability metrics in a user-friendly manner. The UI/UX has been improved to provide
a clean and smooth user experience. All features have been implemented, tested, and are fully functional
for demonstration.

## Features Implemented
- OTP-based authentication for secure login
- Readability Dashboard with visual metrics
- Improved UI/UX design
- Fully tested application

## Tech Stack
- Python
- Flask / Streamlit / Django (your choice)
- HTML, CSS, JavaScript
- GitHub
- Google Colab

## How to Run
1. Clone the repository
2. Navigate to Milestone2 folder
3. Install dependencies using `pip install -r requirements.txt`
4. Run the application

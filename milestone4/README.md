# Milestone 4 – Admin Analytics Dashboard & User Personalization

## Description
In this milestone, the application was significantly enhanced by introducing a powerful **Admin Management and Analytics Dashboard** along with advanced **User Profile Personalization** features. The Admin panel now acts as a centralized control system that allows administrators to manage users, monitor platform activity, analyze system usage through interactive visualizations, and export important data for further analysis.

The Admin Dashboard provides real-time activity tracking, feature usage insights, and graphical analytics to help administrators understand how the system is being utilized. It also includes feedback analysis using **WordCloud visualization**, allowing administrators to identify commonly used terms, user sentiments, and improvement suggestions.

On the user side, the platform now supports enhanced **profile personalization and security management**. Users can update their account information, change passwords, and upload profile avatars to personalize their profiles. Additional UI/UX enhancements were implemented to improve navigation, usability, and the overall user experience.

These improvements strengthen both the **administrative control capabilities** and the **user engagement experience** within the platform.

---

## Admin Dashboard (Management & Analytics)

The **Admin Dashboard** serves as a central command center with the following capabilities:

### User Control
Administrators can manage user accounts through the dashboard by performing actions such as:

- Promote users to **Admin status**
- **Delete user accounts**
- **Lock and unlock accounts** for security control
- View registered user information

### Activity Tracking
The system tracks and displays user activities including:

- Feature usage history
- Model usage logs
- User activity timestamps
- Real-time system activity monitoring

This helps administrators understand how users interact with the application.

### Data Visualization
Interactive visualizations provide insights into system usage, including:

- **Which AI models are being used**
- **Which languages are being utilized**
- **Which features are most popular among users**

These charts help analyze user behavior and platform performance.

### Feedback Analysis
User feedback collected through the system is analyzed and visualized using:

- **WordCloud representation of feedback comments**
- User ratings and feedback summaries

This enables administrators to quickly identify trends and common suggestions.

### Data Export
Administrators can export system data for reporting and analysis, including:

- User details
- Activity logs
- Feedback records
- Deleted user history

All exported data can be downloaded in **CSV format**.

## User Dashboard & Profile Personalization

The user dashboard was enhanced with additional features to improve personalization and account security.

### Security Features
Users are now able to:

- Update their **email address**
- Change their **account password**
- Maintain better control over their account security

### Identity Personalization
Users can personalize their profile by:

- Uploading a **profile avatar (Display Picture)**
- Updating profile information
- Viewing their activity history

### UI/UX Enhancements
Several improvements were made to enhance the interface and user experience, including:

- Modern and responsive dashboard design
- Improved sidebar navigation
- Profile avatar integration
- Better layout and interaction flow

These improvements provide a smoother and more engaging user experience.

## How to Run

1. Install the required dependencies

```bash
pip install -r requirements.txt
```

2. Run the Streamlit application

```bash
streamlit run app.py
```

3. (Optional) Expose the application using Ngrok

```bash
ngrok http 8501
```
